# 🌍 AI Trip Cost Estimator

An intelligent, interactive web application designed to help users plan trips and predict travel expenses accurately. Developed with a premium modern design, the estimator dynamically calculates costs based on seasonality, duration, group size, and specific travel preferences.

## 🚀 Live Demo

**[Check out the live application here!](https://rudrabambal.github.io/AI-Trip-Estimator/)**
*(Note: If you just turned on GitHub pages, it might take a couple of minutes for the link to activate).*

## ✨ Key Features

- **💡 AI-Powered Predictions**: Calculates expected costs dynamically based on the specific inputs rather than generic estimates.
- **📅 Dynamic Seasonality Calendar**: Uses a built-in date-picker to analyze what time of the year you are traveling (peak season, off-season) to adjust costs accurately.
- **👥 Group Travelers**: Easy-to-use stepper controls for entering the exact number of travelers.
- **🍔 Meal & Accommodation Configurations**: Toggle constraints to customize what kind of trip you want (e.g. Budget vs Luxury).
- **💎 Premium Glassmorphism UI**: Beautiful, fully responsive frontend layout built from scratch to look stunning on both Desktop and Mobile devices.

## 🛠️ Built With

* **HTML5** (Structure)
* **Vanilla CSS3** (Styling, Layouts, and dynamic styling such as Glassmorphism)
* **JavaScript (ES6+)** (Logic, Event handling, DOM manipulation, Cost Calculation Algorithms)

## 💻 Running the Project Locally

If you'd like to run a copy of this project on your own machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/Rudrabambal/AI-Trip-Estimator.git
   ```
2. Open the project directory:
   ```bash
   cd AI-Trip-Estimator
   ```
3. Because this relies purely on static frontend files, simply open `index.html` in your favorite web browser or use VS Code's **Live Server** extension.
